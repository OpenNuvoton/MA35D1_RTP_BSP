◆ ADC_Open()

void ADC_Open ( ADC_T *  adc,
uint32_t  u32InputMode,
uint32_t  u32OpMode,
uint32_t  u32ChMask 
)

This API configures ADC module to be ready for convert the input from selected channel.

Parameters
[in]adcBase address of ADC module
[in]u32InputModeInput mode. Valid values are:
  • ADC_INPUT_MODE_NORMAL_CONV
  • ADC_INPUT_MODE_4WIRE_TOUCH
  • ADC_INPUT_MODE_5WIRE_TOUCH
[in]u32OpModeCould be
  • ADC_HIGH_SPEED_MODE
  • ADC_NORMAL_SPEED_MODE
[in]u32ChMaskChannel enable bit. Each bit corresponds to a input channel. Bit 0 is channel 0, bit 1 is channel 1... This parameter is only used while u32InputMode set to ADC_INPUT_MODE_NORMAL_CONV.
Returns
None
Note
ADC can only convert 1 channel at a time. If more than 1 channels are enabled, only channel with smallest number will be convert.
This API does not turn on ADC power nor does trigger ADC conversion
